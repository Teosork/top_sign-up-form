# top_sign-up-form

A responsive sign-up form for an imaginary service called ODIN.

## Features

- Two-panel layout (image + form)
- HTML5 form validation with patterns
- Visual feedback:
  - Blue border on focus
  - Red border on invalid passwords
- Placeholder text for each field

## Technologies

- HTML5
- CSS3 (Flexbox)

## Form Fields

- First Name (3+ letters)
- Last Name (3+ letters)
- Email
- Phone Number (XXX-XX-XXX format)
- Password (8+ chars with uppercase, lowercase, numbers)
- Confirm Password
